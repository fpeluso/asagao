#### Features
- Timeline
  - Home
  - Mentions
  - Favorites
- Tweet
- Media
  - Twitter official (pictures and videos)
  - Instagram
- Other
  - Add/Remove favorite

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run unit & end-to-end tests
npm test


# lint all JS/Vue component files in `src/`
npm run lint

```
